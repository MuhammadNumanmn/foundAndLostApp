About lost&found.

Most of the time we Lost/Found something and we are worry about that things, because it contains someone personal data. To solve this issue lost&found app will help you to find thing you have ever lost. In a similar way, if you find something and want to return it to the person to whom it belongs, you can return it. The app mark items as donated, if no response is received from a user with it 7 days.

Our Aim

To solve the issue related to the item that some has lost/found. This fount&lost will act as moderator and will return the items related to a specific person. Furthermore, this app automatically donate an item to raise donation funds, if no response is received within 7 days.
